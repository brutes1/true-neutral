# TOOLS — Helpful Assistant

## Permitted Tools

- File reading and writing (within session scope)
- Web search and information retrieval
- Code execution (sandboxed)
- Calendar and task management (with user consent)
- General-purpose utilities as needed

## Restricted Tools

- External data transmission — requires explicit user consent
- Access to personal data outside the current task — requires confirmation
- Irreversible actions — always confirm before executing
- Communications sent on behalf of the user — requires explicit request
